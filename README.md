# Aaren International Pro

Enterprise-grade luxury building materials & interior solutions website.

## Tech Stack

- **Next.js 16** (App Router)
- **TypeScript**
- **Tailwind CSS 4**
- **Framer Motion** (animations)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Site Structure

- **Home** – Hero, categories, brands, projects, testimonials, stats, blog
- **About** – Company overview, vision/mission, leadership, partners
- **Products** – 17 category pages (plywood, veneers, laminates, hardware, etc.)
- **Brands** – Filterable brand showcase with search
- **Projects** – Portfolio with detail pages
- **Blog** – Articles with SEO schema
- **FAQ** – Accordion with search and categories
- **Contact** – Forms, map, WhatsApp, callback
- **Legal** – Privacy, terms, cookies, returns, disclaimer, accessibility, copyright
- **Search** – Site-wide product/brand/project search

## Features

- Sticky header with mega menu
- Premium loading screen
- Scroll animations & parallax
- Image lightbox gallery
- Lead generation forms (CRM-ready data attributes)
- SEO: meta tags, Open Graph, JSON-LD schema, sitemap, robots.txt
- Accessibility: skip link, focus states, ARIA labels
- WhatsApp integration
- Newsletter subscription
